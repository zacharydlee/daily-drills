# Flexbox Buttons

A small project focused on formatting multiple buttons and elements using flexbox.

## 🎯 Purpose

To practice:
- structuring a webpage using basic HTML
- Organizing layout with Flexbox (column and row systems)
- Applying CSS box model for spacing and sizing
- Building simple interactive UI components (buttons). 

## 🏗 Features

- Centered page layout using Flexbox
- Verticle layout for page structure
- Horizontal layout for button grouping
- Stylized buttons with:
  - padding and borders
  - rounded corners
  - box shadows
  - hover and active states

## 💡 Concepts Practiced

- Flexbox (display: Flex, flex-direction, justify-content, align-items, gap)
- Box Model (margin, padding, border)
- CSS selectors and psuedo-classes(:hover, :active)
- Basix UX/UI styling principles
