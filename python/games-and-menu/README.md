## Python CLI Game Hub

This is a simple command-line game hub built in Python. 

## 🎮 Games Included
- Number Guessing Game (Guess My Number)
- Rock, Paper, Scissors

## 📊 Features
- Menu-based CLI interface 
- Input validation
- Game loops for both games
- Modular file system

## 📚 Skills Practiced
- Python functions and control flow
- User interaction and input validation
- Modular program design

## 🚀 How to Run
```bash
python menu.py
```

This is a work in progress. I plan to expand the game hub with additional games, over time.
