# Number Guessing Game

A command-line number guessing game built in Python

## Description 

The program selects a random number on the player's chosen difficulty level.
The player must guess the number within 5 attempts. 

## Features

- Difficulty selection:
  - Easy (1-10)
  - Standard (1-25)
  - Hard (1-50)
  - Nightmare (1-100)
- Input validation for all user inputs.
- Limited number of attemps per round (5).
- Feedback on guesses ('too high' / 'too low'
- Replay functionality

## How to Run
```bash
python guess_the_number.py
```
